# Counter Service Scheduling System

ระบบจัดตารางเวรสำหรับเคาน์เตอร์บริการ (Counter Service Scheduling System)

## Features

- **Schedule Generation**: Automated shift scheduling for Tha Phra Chan (TP) and Rangsit (RS) locations.
- **Staff Management**: Manage staff details, roles, and availability.
- **Calendar View**: Visual calendar for viewing schedules.
- **Export**: Export schedules to PDF (if implemented) or view printable summaries.

## Getting Started

### Prerequisites

- PHP 7.4 or higher
- MySQL or MariaDB
- Apache/Nginx web server

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/wptu/counter_service.git
   ```
2. Import the database schema from `php/database.sql` or `php/init.sql`.
3. Configure your database connection in `php/src/config/database.php`.
4. Run the application on your local server.
